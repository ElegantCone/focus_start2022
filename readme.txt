Версия java - 11.0.14
Версия maven - 3.8.4
Для сборки jar файла из папки с pom.xml: mvn clean compile package
.jar появится в папке target.
Для запуска программы из командной строки: java -jar FOCUS_START-1.0-SNAPSHOT.jar [-a|-d] -i|-s output_file input_files
Файлы должны располагаться в одной папке вместе с .jar, либо на вход должен подаваться полный путь расположения файла.
Выходной файл переписывается при каждом запуске программы и может не существовать.
